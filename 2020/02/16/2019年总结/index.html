2019年总结

发表于 2020-02-16 更新于 2022-03-20
2019之总结与展望

总结

疫情赋闲在家，忽然想起荒废两年多的blog，当初凭一时热情写了几篇博文，随后便因为生活工作上各种各样的事导致更新搁浅，等到平静下来却又没继续下去的动力了。不过毕业后接近三年，心境早已和以往大不相同，个人的目标也渐渐明确，恰好一年伊始，总结一下过去动荡的19年，顺便展望下20年，也当是激励自己再起写技术博文的习惯吧。
19年于我而言也是颇为印象深刻的一年。“疲惫”应该算是比较适合我的19年关键词。自18年9月底从第一家公司裸辞后，也是因为身体和精力问题，大部分时间都在家中休息，长时间的颓废也导致我停下了学习的脚步，和同龄人的差距一步步拉大，这也成了我之后很长一段时间焦虑的原因。
随后，2月底终于确定了手术流程，我也算是摆脱了先天疾病带来的阴影，还获得了ICU一日游的“成就”。修养恢复，4月开始在杭州开始找工作，近半年没有学习工作也确实令我有些落后于技术潮流，加上我和杭州实在相性不佳（16年在杭州实习就因为没有HC无法转正，找工作时小公司也不看重我曾经的工作方向），四处碰壁，最低落时甚至准备降薪5K去一家传统行业，然后来考虑了自己的未来规划，还是在入职当前去公司现场拒绝了offer。
再后来的事情就流水账一下吧，6月拿到上海创业公司offer，又回到一线城市打拼；被想去杭州的“原因”拒绝，再次体会到感情这种事可遇不可求；9月参加同学婚礼，觉得自己比同龄人慢了半拍；10月因为强度加班，休息不当开始持续耳鸣，身体小毛病越来越多，目前为止还没有好转；跨年时和相识多年朋友湖南旅游，又因性格问题闹的很僵不欢而散，反思之后觉得自己容易走极端的习惯和幼稚行为确实要改改了。
工作方面，在新公司自身的工作重心转到了业务上，繁忙的业务导致业余时间根本没有余裕去学习技术或者刷题。不管是技术能力，还是薪资水平，都没有达到预期。唯一让自己满意的就是在新公司做业务需求之余自发做了不少优化，封装，组件化开发相关的工作。
自我感觉19年的经历可以总结成一句话————经历的不少，却并没有做好。

展望

春假，从繁忙的工作中抽离开，沉静下来思考的时候才会知道自己真的想要的是什么，我也很清楚自己目前还是太过年轻，对所谓的胜负有比较强的执念，这即是我倍感焦虑的原因。既然心境不是一思一考之间就可以改变，那还不如利用自己的执念让自己有所进步。
学习，我希望这能成为我20年的关键字，在此也立下几个目标，看来年能完成多少。

列出以下几项，希望至少能成体系的进行学习，括号当中选择一个即可
Android方面：

  • 常见架构实践，最好有相关文章产出
  • 组件化技术沉淀
  • 性能优化，性能监控，相关demo或者项目中应用
  • 未涉猎区域（跨平台Flutter，音视频）
  • native

计算机基础

  • linux，相关工具
  • （操作系统，计算机网络）

其他：

  • 前端入门
  • 了解一些常用后端框架
  • 读书，读书，读书
  • 锻炼身体（身体是一切的基础，小毛病频发让我更深刻认识到这点了）
  • 思考方法论

希望不是FLAG，姑且对自己说一声加油。
